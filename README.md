# 🌐 My Portfolio Web

This is the structure of my personal portfolio. It includes multi-language support and is divided into clean, reusable components.

## 🛠️ Tech Stack

*   **Vite** (Build tool)
*   **React 19** & **TypeScript**
*   **Tailwind CSS v4**
*   **Framer Motion** (For smooth animations)

## 🚀 Installation & Usage

Follow these steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/StubbornRuby763/Portafolio
   cd Portafolio

    Install dependencies:
    Bash

    npm install


3. **Run the development server:**
   This project uses `npm-run-all` to compile Tailwind CSS and launch Vite simultaneously:
   ```bash
   npm run dev
   

    Build and Preview for Production:
    To compile TypeScript and build the static files:
    Bash

    npm run build

   To preview the production build locally:
   ```bash
   npm run preview
